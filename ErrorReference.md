## 前缀类型说明

| 前缀  | 说明         |          默认HTTP状态码          |
  |:---:|:-----------|:---------------------------:|
| `F` | 服务器异常      | `500 Internal Server Error` |
| `D` | SQL数据库异常   | `500 Internal Server Error` |
| `G` | MongoDb 异常 | `500 Internal Server Error` |
| `R` | Redis数据库异常 | `500 Internal Server Error` |
| `I` | IO 过程异常    | `500 Internal Server Error` |
| `P` | 数据转换时异常    |      `400 Bad Request`      |
| `C` | 数据校验异常     |      `400 Bad Request`      |
| `S` | 资源查找异常     |       `404 Not Found`       | 
| `M` | 数据未变更      |     `304 Not Modified`      |
| `A` | 权限认证异常     |     `401 Unauthorized`      |
| `Q` | 七牛云上传异常    |      `400 Bad Request`      | 
| `L` | 日志报告异常     | `500 Internal Server Error` | 
| `T` | 腾讯云服务异常    | `500 Internal Server Error` |
